Description

This website is the result of a project where we set out to provide an easy-to-use carbon footprint calculator for importing coffee beans into the United Kingdom. The results of this calculation allow coffee bean roasteries to more accurately offset their carbon emmisions. The website was built in conjunction with Glen Lyon Coffee Roasters by students of Edinburgh Napier University. The source code for the website can be found at https://github.com/lewisstevenmackenzie/Glen-Lyon-Group-Project .

How to Use

Instructions

Non-logged in Users:
• Navigate to the calculator page.
• Select the origin country from the drop down menu.
• Input the distance from the coffee plantation to the Port (km).
• Insert the distance from Tillbury Docks to the UK Roastery (km).
• Finally input the total weight of the package (kg).
• Now click the submit option and the carbon emissions will be displayed below in kg.

Logged in Users:
• Log in and select add route.
• Choose a title for the current calculation.
• Select the origin country from the drop down menu.
• Input the distance from the coffee plantation to the Port (km).
• Add a name for the end location.
• Insert the distance from Tillbury Docks to the UK Roastery (km).
• Finally input the total weight of the package (kg).
• Now click the submit, this will now post the calculation to the users home page.
